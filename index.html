OEE Formulas

Equations used in this system

OEE Formula

OEE = Availability × Performance × Quality
Example:
A=90%, P=75%, Q=98%
OEE = 0.90 × 0.75 × 0.98 = 66.15%

Availability

Availability = Run Time / Planned Production Time
Run Time = Planned Time − Unplanned Downtime
Planned Time = Shift Length − Breaks
Example:
Shift=720, Breaks=120, Downtime=55
Planned=600, Run=545
Avail = 545/600 = 90.83%

Performance

Performance = (ICT × Total Count) / Run Time
ICT = 1 / Ideal Rate. Capped at 100%.
Example:
Rate=135/min, ICT=0.00741
Count=65000, Run=545
Perf = (0.00741×65000)/545 = 88.35%

Quality

Quality = Good Pieces / Total Pieces
Example:
Total=65000, Rejected=1400
Good=63600
Q = 63600/65000 = 97.85%

3-Stage Production

Our production has 3 stages:
Extruding Printing Cutting

Stage OEE = Average OEE of all machines in that stage

Product OEE = Stage1 × Stage2 × Stage3 (when product passes through all)

Each product uses 3+ machines (one or more per stage).
Example Product PRD-001:
Extruding (BM-EX01): 88%
Printing (BM-PR01): 75%
Cutting (BM-CU01, BM-CU02): 82%
Combined = 0.88×0.75×0.82 = 54.1%

Multi-Product / Speed Changes

When machine runs multiple products with different speeds:
Weighted ICT = Σ(ICT × Count) / Σ(Count)
Example:
Product A: 135/min × 40000
Product B: 110/min × 25000
= (135×40000 + 110×25000)/65000
= 125.4 pcs/min average
Enter each product run as a separate production entry.
